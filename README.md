## Run code in console

Open console and click/tap the Run button.

Depends on the following HTML:

```html
&lt;pre&gt;
	&lt;code&gt;
		&lt;!-- example JavaScript with console.log call --&gt;
	&lt;/code&gt;
&lt;pre&gt;

&lt;button&gt;&lt;/button&gt;
```
